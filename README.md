# Student Marks Analysis

This is a beginner data analysis project built using Python and Pandas.

## Project Description
The project analyzes student marks in different subjects and calculates:

- Average marks of each student
- Top performing student

## Technologies Used
- Python
- Pandas

## Files
- student_marks_analysis.py → Python script for analysis
- student_marks.csv → Dataset of student marks

## Output
The script calculates averages and identifies the top student.
